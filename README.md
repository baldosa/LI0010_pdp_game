### Run

Requisito Python >= 3.10

1. Crear y activar virtual env

```bash
python -m venv venv
source venv/bin/activate
```

2. Instalar requisitos

```bash
pip install -r requirements.txt
```

3. Ejecutar

```bash
python run.py
```

4. Se mueve con WASD y se dispara con left click
